` ip <moznost> <akce> 


-  `<ip r>` - route 
- `<ip a>` - address
- `<ip r default>` - default internet connection
- `<ip link>` - prace s fyzickymi porty
- `<ip a show dev eth0>` - ukaze konfiguraci eth0 
- `<ip nei>` - ukazuje jaka ip ma jake mac 
- `<ip nei replace <ip> lladdr <mac> dev <interface> nud permanent` - prenastavi mac u ip pokud uz existovala, pokud ne tak `replace --> add`
